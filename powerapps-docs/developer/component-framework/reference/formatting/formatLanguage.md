## Available for

## Syntax

`context.formatting.formatLanguage(value)`

## Parameters

| Parameter Name|Type|Required|Description|
| ------------- |----|--------|-----------|
|value|`number`|yes|A language code to be formatted.|

## Return Value

Type: `string`
